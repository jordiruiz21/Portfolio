PeliAPI

Mis Pelis es una aplicación web desarrollada con React y Bootstrap que permite a los usuarios explorar y gestionar su propia colección de películas favoritas. La página principal muestra una lista de películas populares obtenidas mediante una API externa, con la posibilidad de realizar búsquedas personalizadas por título. Este proyecto me sirvió para poner en práctica lo sabido en cuanto al uso de API´s y React

Tecnologías utilizadas

HTML CSS JavaScript React Bootstrap
