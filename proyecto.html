FitCore

FitCore es una plataforma web desarrollada con Django para la gestión de un gimnasio, con registro de usuarios, rutinas de entrenamiento y seguimiento del progreso mediante gráficos. Permite crear, editar y eliminar ejercicios, músculos y centros. Fue un proyecto en el que aprendí mucho tanto del frontend como del backend.

Tecnologías utilizadas

HTML CSS JavaScript Django Python
